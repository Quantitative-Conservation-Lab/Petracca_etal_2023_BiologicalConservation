IPM_data_collection_period.R -- has full IPM summarizing wolf population from 2009-2020 (data collection period)

Projection_model_leastcost_territory_selection.R -- has spatial projection model for projection period (2020-2070) using categorical RSF territory selection process

Projection_model_RSF_territory_selection.R -- has spatial projection model for projection period (2020-2070) using least cost path territory selection process

The folder "Preliminary_Analysis_Steps" contains four R scripts to (1) identify dispersal movements, (2) calculate mean territory size, (3) perform Resource Selection Function, and (4) identify the "best" territory at multiple distances from each territory using the RSF output
